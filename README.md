# Projek-Basis-Data-Kelompok-G
Proyek ini menerapkan konsep Basis Data Relasional pada studi kasus MarketPlace/E-commerce. Dikembangkan secara kolaboratif via GitHub, cakupan proyek meliputi analisis kebutuhan, perancangan ERD, normalisasi, implementasi DDL/DML, hingga pembuatan fitur lanjutan seperti query kompleks, view, stored procedure, dan trigger.
